Glossary
=============================================
.. glossary::
   SID
      .. seealso:: :doc:`sid`
      | ID generated when logging into Amino, containing the User ID, IP address, device type and some keys.
      | SID authenticates requests at Amino.

   Device ID
      .. seealso:: :doc:`did`
      String that uniquely identifies a device, valid ones start with 01. Used to check if requests are made from a real device.


   ComID
      ID that identifies communities, used in functions to target specific communities.
